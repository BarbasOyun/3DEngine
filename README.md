# 3D Engine

-Using Rust + egui<br>
-Using [This Tsoding video](https://www.youtube.com/watch?v=qjWkNZ0SXfo&list=WL&index=9&t=17s) as reference<br>

## Features

-Render 3D Models Wireframe<br>
-Transform 3D Model<br>
-Transform Camera<br>
-Import OBJ File<br>

## Learnings

-Linear Algebra<br>
-Camera System<br>
-Rendering Pipeline<br>
-Graphics Programming in General<br>

-A lot of dead code but I want to keep it as exemple (Old Engine)<br>
